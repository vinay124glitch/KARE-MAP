# KARE Campus Map - Real-Time Navigation

A premium, interactive 3D campus map web application built for **Kalasalingam Academy of Research and Education (KARE)**. This application features real-time GPS tracking, heading detection, and 3D building visualization.

## âœ¨ Features

- **3D Interactive Map**: Explore the KARE campus with 3D extruded buildings and custom styling.
- **Real-Time Geolocation**: Tracks your exact position on campus using the browser's Geolocation API.
- **Heading & Direction**: The map and user marker rotate in real-time based on your device's compass/orientation.
- **Navigation**: Search for campus buildings and get instant paths from your current position.
- **POIs**: Clickable points of interest with detailed descriptions.
- **Glassmorphism UI**: Modern, premium user interface designed for both desktop and mobile.
